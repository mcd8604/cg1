Platform on which you tested your implementation: 
	
	Windows XP, Visual Studio 2008



Instructions for building and running:

	Open Assignment2.sln, Build and run using Visual Studio 2008.

		OR

	Run 3DDrawing.exe from the 3DDrawing directory.

	Press a or z to start animation.



Keyboard Input:

	a: Increase rotation speed

	z: Decrease rotation speed

	c: Change cameras

	1-4: Toggle lights 1-4
	
	q: Quit program

	s: Add Ring

	x: Remove Ring
